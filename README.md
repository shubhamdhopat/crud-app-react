# CRUD App with React, Vite, and Axios

A simple CRUD web application built with **React**, **Vite**, and **Axios**.
This project demonstrates how to perform **Create, Read, Update, and Delete** operations using a REST API with modern React hooks and component-based architecture.

---

## рџљЂ Features

* Fetch posts from a REST API
* Display posts in a structured list
* Add new posts using a form
* Edit existing posts
* Delete posts instantly
* Axios integration for HTTP requests
* Clean and responsive UI
* Reusable React components
